## Weather Forcasting
 
Flask MicroFramework build a weather forcasting application system

## Features 

- User Form Checking
- Analysis daily day of weather
- Render Template ui
- PostreSQL + SQLALCHEMY

## Setup

1. cp .env.development.local  .env.local
2. Update .env.local with your DB URL and secret key
3. pip install -r requirements.txt
4. .gitignore
5. run python flask application
